# Realtime Poll App

Live polling application built with the MERN stack and Socket.IO.

## Features

- User authentication
- Poll creation
- Real-time voting
- Anti-duplicate voting
- Persistent storage

## Tech Stack

- React
- Node.js
- Express
- MongoDB
- Socket.IO
- TailwindCSS

This is MERN Project full developed in One Day
Allows you to create poll and share with the others to votes.
You can see the older polls on user Dashboard

Setup:

- fork and Clone the Git Repo on your Local system.
- create .env files and put the enviroment variables for client and server side both.
- /server/.env - PORT - MONGO_URI - JWT_SECRET
- /client/.env - VITE_API_URL : frontend url port ex : localhost:1234

- npm install in both folders server and client
- npm run dev in both folders server and client
